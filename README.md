# autoconverter-mkv-mp4
Auto converter for all files.mkv to files.mp4 in your folder

## Install guide
Install dependency
```bash
pacman -S ffmpeg
```
Install **autoconverter-mkv-mp4**
```bash
git clone https://github.com/holovin777/autoconverter-mkv-mp4.git
cd autoconverter-mkv-mp4
python autoconverter-mkv-mp4.py
```
